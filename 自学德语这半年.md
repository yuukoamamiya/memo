# 自学德语这半年
我从 2020 年 7 月开始学德语。除了寒暑假以外，只能在课业之外抽零碎的时间学习。去年九月是我第一次参加歌德 B2 考试，近乎裸考，有幸高分通过。

2020 年暑假，我脱产学了两个月，接下来的秋季学期断断续续上了十次一对一的课。去年寒假，我报了一个 B2 脱产班，发现班上的同学差不多只有 A2 的水平，有些拖慢进度，于是下决心开始自学。寒假，我脱产自学了两周，去年春季学期保持了每天半小时的自学时间。去年暑假，我报名了歌德学院的德语课程，班上的同学几乎都学习了至少两年，而我的德语已经比他们所有人都好。

P.S. 我的自学方法不仅适用于德语，对于其他外语的学习也是通用的，已经有朋友把我的方法实践到了英语学习上，取得了非常不错的效果。

## 为什么要自学一门外语？（什么样的人适合自学？）

跟着班级进度学习一门外语无疑是更轻松方便的，不需要自己设计教学方案和把控学习的进度。但是，虽然德语教学机构的老师们或许是经验丰富的，但是，在经验之中对教学原理进行反思，又是另一个层面的要求。

有的老师对人类学习和记忆的原理理解并不深入，哪怕在德国接受了 “德语教学法” 的科班教育，不乏有人倡导背诵课文这种事倍功半的方法。在我的经验里，只有歌德学院对大课教学的设计令人满意，不论是教材选择、课堂组织，还是技术设施的配合（包括 zoom、从德国出版社买的电子教参、词汇卡片、单词搭配小游戏）。对于有的老师，优秀的德语水平并未赋予他们对等的资源收集能力，很多人在收集资源时仍然中德夹杂着进行百度搜索。英文互联网和德文互联网上有非常丰富的语言学习资源，但这都被阻隔在墙外。

当然，不仅德语老师们很难满足上述要求，这对于一个普通人而言也很难。如果决定要自学一门语言，你需要做好心理准备：初期你需要花大量的时间收集资源和设计教学法，并且学会用技术手段实现这些教学安排——虽然不需要代码基础，但的确需要现学现用少量相关知识；小众开源项目的粗糙性，也需要你不断地在电脑前 trouble-shooting。我试图辅导不少朋友自学语言，最后真的完成了教学设计的人只有一个 CS 科班出身的同学。不过不需要特别担心，我刚开始自学德语的时候也不懂这些，在探索的时候，我发现需要掌握一些编程语言，很快就现学现用了起来；不管怎样，你得是一个不反感折腾电脑的人。

除了自己设计教学的麻烦事儿以外，自学总还是孤独的，对自己的进度也很难心里有数。半年以来，我一直感到惶惶不安，担心我这么胡搞影响了德语学习的进度。直到在歌德学院上课，我才意识到自学让我的德语进步这么快。虽然暑假的课是在线上，我在德语班上认识的朋友，也终于让我跳脱出之前的自闭生活。当然，这也有赖于歌德学院的分级考试机制，水平差太多的同学是不会被分进同一个班的，否则，被其他同学拖慢进度带来的焦虑，远远多于社交带来的快乐。

当然，我们可以用各种方法来给自己 “进度感”，比如制定计划和规定每日打卡目标。21 年春季学期，多亏了我的打卡群，我才能坚持每天学习 0.5–1 小时；刚开始自学的时候，我更是直接给导师打了一个寒假的卡，比起给同学打卡或者金钱激励，还是导师的期望更能鞭策我 orz

如果能克服上述这些困难，自学语言是好处颇丰的。是否要选择自学，需要考虑自己的情况来决定。

## 使用语言和间隔记忆

在我粗浅的理解中，学习一门语言，最重要的是使用它，用它去读、听、说、写。母语者怎么用，你就怎么用。在自学过程中，首要的就是每周都读了、听了、说了、写了一些东西。

背单词、背课文、记忆语法规则、学习语言学等方式，都背离了这一宗旨。背单词是把单个词汇挑出来、曲解它、把它对应上母语里的一个单词，然后通过简单重复让我们的大脑在二者之间建立联系。背课文也是简单粗暴的记忆和发音练习，同主动口语表达完全是两回事。语法讲解则是试图把具体的句子形态归纳成普遍的结构。

我们当然还是可以借助这些方式学习，只是不能本末倒置。之所以要在上一段贬低这些方式的作用，是因为我见到太多的同学把绝大部分语言学习的时间花在了这上面。流水一般的外语材料输入输出，如果达到一定的量，足够把我们训练为母语者，但是，已经成年的我们，不再有大把大把的时间，放在游戏或基础教育上，只能在正事儿中挤时间学习语言。假如我有时间在不同语境中见到同一个单词 100 次，我相信我对它的理解只会比把词典释义背诵下来更好，在它出现的时候，我也能更自如地回忆起它的含义。可惜我没有这么充裕的时间。语法点的学习和相应练习也是同样的道理。已经成年的我们比孩童更擅长抽象思维，句子骨架即使没有具体内容也可以轻易记住；我们也能更方便地借助语言学的研究成果辅助学习记忆。总之，没有足够的时间在一种语言中游戏与生活，我们只能借助成年人的高阶能力来加快语言学习，采用背单词、记忆语法规则、学习语音学等方式。

那么，我们应该怎么样背单词、学语法呢？除了对每项任务的具体安排（比如要在语境中背单词，我会在后面详细讲），我感到一个关键是间隔记忆。我精读、精听过的材料，都会把单词、表达和语法点做出 anki 卡片，按照 anki 的算法在预计好的时间背诵。知乎上已经有很多文章科普用 anki 学语言的好处了（传送门：[Anki 怎么用？](https://www.zhihu.com/question/28335314/answer/428970652)），于我而言，最大的几个好处是

-   很多背单词 app 喜欢吹艾宾浩斯记忆曲线，但论记忆算法还是都比不上 anki 和 supermemo。
-   anki 开源社区有很多 add-ons，保证我每天都可以很快做出卡片。
-   我也会精心设计卡片布局，保证这些卡片符合我的记忆习惯。比如说，一张卡片的核心并非单词，也不是释义，而是例句。对我来说，背卡片的意义远远大于 “在外语词汇和母语词汇之间建立记忆关联”，而是反复熟悉这一语言中的众多小语境。
-   把学习内容转化为 anki 卡片，也更容易给自己 “进度感”。我给自己规定好，每天都做一定量的新卡片，并且按时完成记忆算法给我安排的复习任务。这对于自学是非常重要的。

对于 anki 的用法，后文涉及到具体的学习内容时，我会详细展开。

## 发音

学习德语，首先要学的就是发音规则。尽管德语教学机构里留德回国的年轻老师发音多还不错，但我感到只有极少数老师懂得如何教授发音技巧。德语的发音是非常难的，一方面，高密度辅音很考验口腔肌肉的灵活度，三个辅音连击的现象都不算罕见（比如 geimpft，“已接种的”）；另一方面，德语的很多音素是中文和英语里都没有的，除了老生常谈的小舌音以外，还有 ö, l, ch 等等。

很多老师都会说，发音不重要，别人能听懂就行。但是，

-   一方面，发音会严重影响表达速度。很多同学学了两三年德语以后，这种影响还是非常严重。他们说德语时，常常做的是费力地探索自己的口腔，然后绝望地选了个相近的发音部位，然后奋力地卡口水。
-   另一方面，某些口音会严重影响可理解性。比如说，对于 will 这个单词，很多南欧人可能会发成 wi-le，这个发音不会引起任何歧义，le 只是把半元音 l 发成了辅音，但发音部位都在牙后的 alveolar ridge。然而，很多中国人会把 will 发成 wir，发音部位都变了，是很容易让别人听不懂的。

如何学习发音规则？对口腔肌肉的高要求，自然需要你多加练习。一开始没办法顺利启动特定的肌肉很正常，重要的是不要糊弄过去，一开始可以慢一点，但每次开口都要尽力说标准，这样才能起到练习的效果。

除此以外，语音学的知识也大有裨益。关于语音学的一般知识，推荐学习[A Guide to Pronuncation in Any Language by Fluent Forever](https://www.youtube.com/playlist?list=PLlmgeS8moU5ggCOvV3x8MHZy7vY-wucdU)和[Phonetics on Wikipedia](https://en.wikipedia.org/wiki/Phonetics)；关于德语语音学，推荐学习[German Pronuncation by Fluent Forever](https://www.youtube.com/playlist?list=PLvyVfn_eZOHoiOPk09XOTlOAdHYEzz5JW)。关于发音部位（见下图）、发音方式（塞音 t、擦音 s）的知识，能够辅助你更准确地发音。尤其是对于中文和英文里没有的发音方式，例如 l 和 ö，你可以明确知晓应该努力调动哪块肌肉。

![](https://img2.doubanio.com/view/note/l/public/p87083863.jpg)

除了语音学以外，听德语歌也对发音有很大帮助。音乐的编排导致某些音节拖得很长、发得很夸张，让人清楚地听到发音过程。例如，Egal, was soll's 在我看来就是最好的响音 / l / 教程（让你知道为啥 / l / 会响 23333，虽然在一般理解里它是一个辅音）。我之前写过一篇文章，分享了自己[听德语歌做的 anki 卡片组](https://zhuanlan.zhihu.com/p/374618448)，可以参考。

## 语法

-   [新求精初级语法教材](https://book.douban.com/subject/25807876/)，基本上按照初级 I 和初级 II 两本书的章节结构安排。目录只写了第几章第几节，不方便查知识点，更适合配合教材学习。这本教材会对语法点做很多细微的区分，现在看来并无必要。德语语言大的精神又没有把握住、需要自己去体会。 我在学习过程中完全离不开 wikipedia 和 wiktionary，在它们的帮助下能够大致体会到德语语法的精神。但一起学习的很多同学并没有做到这一点，因此哪怕做了再多练习，语法还是有些问题。这其实是教材和教学的问题，比较可惜。

## 怎么读和听？用什么读和听？

B2 及以前的阶段，课本仍然是主要的输入材料。我没有用过太多课本，基本的感受是，原版教材都还不错。选教材时可以参见[友邻阿卡西亚的推荐](https://www.douban.com/note/756170217/)和[我的德语教材豆列](https://www.douban.com/doulist/148061314/?sort=time&start=0#item1262064515)。因为我需要结合使用教材和 anki，因此，选择教材时，我的首要考虑是，出版社是否提供了电子教材，这样我才好做单词卡片。

我从 B2 开始使用的是 Aspekte 系列：

原版教材，听读材料都非常地道，表达准确，几乎所有单词或搭配都能在词典上找到 reference。语法的讲解和其他原版教材一样，偏实用，但不够学术，需要搭配 wikipedia 和其他语法讲解。

虽然 libgen 可以下载到，但还是推荐[在 klett-sprache 官网购买电子教材](https://www.klett-sprachen.de/aspekte-neu-b2/t-1/9783126050241)，[klett-sprache 官网还可以下载到相当多的资源，如听力材料的 Transcript](https://www.klett-sprachen.de/aspekte-neu/r-1/15#reiter=mediathek&dl_niveau_str=B2)。

我精读教材遵循一个固定程式：

-   首先通读全文，每句话最多读两遍，标注不明白的地方。
-   再次浏览一遍，主要关注每个段落的开头结尾，理解文章的结构和逻辑。
-   用 Marginnote 3 勾画不明白的句子，查询并记录生词和没有见过的表达（使用之前提到的词典资源），效果如下

![](https://img9.doubanio.com/view/note/l/public/p87083866.jpg)

-   从 Marginnote 3 的笔记导出 anki 卡片。导出方法见[我做的教学视频](https://www.bilibili.com/video/BV1qf4y1a7Ej/)。效果如下

-   把卡片加入 anki，按照 anki 的记忆算法进行间隔复习

这样精读下来，不仅能够提高理解单个句子的能力、理解篇章结构的能力，还能不断熟悉一个个小语境（做成 anki 卡片的那些小语境）。

B2 及以上，泛读必不可少。推荐使用 kindle。kindle 上有生词本功能，会把你记录的生词和该词所在的语境存储在`/[Kindle Volumn]/system/vocabulary/vocab.db`路径里。[我写了一个 anki 插件](https://ankiweb.net/shared/info/443244854)，可以把这个文件转换为 anki 卡片组。这样，泛读时积累的表达，也可以加入间隔记忆的队列中，也让自己有统一的指标来衡量自己的进度。

精听的程式如下：

-   不间断地播放一遍听力材料，没听懂的先不管，尽量跟上大致的意思。听完一遍后，回想一下这篇材料的结构。
-   播放第二次，跟不上时可以暂停、回想一下。
-   播放第三次，如果还有听不懂的句子，可以倒带回放。听完三遍后，再次回想这篇材料的结构。
-   在 Marginnote 3 中打开 Transcript，查询并记录生词和没有见过的表达（使用之前提到的词典资源），效果与精读类似。
-   从 Marginnote 3 的笔记导出 anki 卡片。导出方法见[我做的教学视频](https://www.bilibili.com/video/BV1qf4y1a7Ej/)。效果与精读类似。
-   把卡片加入 anki，按照 anki 的记忆算法进行间隔复习。
-   不间断地播放一遍听力材料。如果还有听不懂的地方，可以回顾一下 Transcript，并重新播放一次。如此循环，直到完全听懂。
-   再次播放听力材料，每播放两三句暂停下来，用自己的话或者听力原句复述这两三句话的大概意思，如果感觉有困难，倒带回放这两三句话，重新尝试复述，直到成功复述。

这样精听也可以提高理解单个句子和篇章结构的能力。与此同时，刚刚听到的东西，也在复述时转化成了主动表达库，是对口语的训练。

## 怎么说和写？用什么说和写？

口语一向是中国学生的老大难，网上一直流传各种提高英语口语的 “秘笈”，比如背课文、复读听力材料。但是在我看来，这些方法效果都一般。背诵和复读，与其说是表达训练，不如说是发音训练。在这一过程中，我们并未尝试主动说出自己想表达的东西。

关键还是足够量的主动表达，尤其是自由表达 Freisprechen。

-   之前提到了复述听力材料，在这一过程中，我们用上自己的话或听力材料里已有的表达，还原听力材料的意思，不要求、甚至应该避免背诵原句。
-   除此以外，我训练口语的方法就是在[italki](https://www.italki.com/)上找德国人聊天。我的德国老师并未提供系统的教学，与其说她是老师，不如说是语伴。我们每次选定一个话题，但具体聊什么内容其实比较随意。

除了自由表达以外，我唯一的刻意训练就是课后复盘。在我表达的过程中，我的老师会及时纠正我的错误表达和奇怪说法，发在 skype 聊天框里，我课后会把这些内容整理成挖空卡片或英译德卡片（给出英文句子，要求翻译出德语）。

复习挖空卡片时，我需要根据语境回想出挖空处的单词，把英语句子翻译成德语句子时，我也需要回想应该用什么德语表达。这种过程与主动表达接近。如果做成普通卡片，我可以很轻松地理解这个单词的意思，但这不代表，当我需要使用被挖掉的单词时，我可以顺利选到它。

训练写作时，我会花很多时间查词典和语料库，确保德语里确实有自己用的这些表达。写完后，我会把作文发给德语老师或者 italki 上的德国人修改。最后，我会复盘自己查到的新表达和他们的修改意见，做成挖空卡片和翻译卡片，与口语卡片类似。
